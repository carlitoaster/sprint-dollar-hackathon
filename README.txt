Run this on your laptop:

1. Install packages
   pip install -r requirements.txt

2. Start the app
   python app.py

3. Keep that terminal open.

4. On another device on the same Wi-Fi or same phone hotspot, open:
   http://YOUR-IP:5000

Notes:
- YOUR-IP will look like 192.168.x.x or 172.20.x.x
- If the page still will not load on another device, your firewall or hotspot is blocking local connections.
- On Mac, allow incoming connections for Python when prompted.
- On a phone hotspot, some devices block client-to-client traffic. Regular Wi-Fi usually works better.
